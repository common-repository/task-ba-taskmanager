== Description ==

First I developed this Tool in german. In a few weeks you can get this Tool in english.

Mir fehlte ein ordentliches Aufgabentool. Deshalb habe ich den Task BA Taskmanager entwickelt.

Der Task BA Taskmanager hat folgende Features:

*   Erstellen von Aufgaben
*   Delegieren von Aufgaben
*   Jeder registriere User kann Aufgaben erstellen
*   Es können Gruppenaufgaben erstellt werden
*   Einstellung, ab welchen Rechten Gruppenaufgaben erstellt werden können. 
*	Einstellung, ab welchen Rechten Aufgaben delegiert werden können. 
*	Erinnerungs-E-Mail bei fällige Aufgaben
*	Einstellung, wann die Erinnerungs-E-Mail 


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `task_ba_taskmanager.zip` to the `/wp-content/plugins/` directory an extract this archive.

OR

1. Download the Plugin from the respority an activate it in the Plugin-Manager.
